# BetterPM

BetterPM 是一个强大的 Bukkit 插件管理器，提供了更好的插件管理功能。它允许服务器管理员在不重启服务器的情况下加载、卸载、启用和禁用插件。

## 功能特性

- 动态加载插件：无需重启服务器即可加载新插件
- 安全卸载插件：正确卸载插件，避免内存泄漏
- 插件状态管理：支持启用/禁用插件

## 安装说明

1. 下载最新版本的 BetterPM.jar
2. 将文件放入服务器的 plugins 目录
3. 重启服务器或使用插件加载命令
